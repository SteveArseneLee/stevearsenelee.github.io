## 함수
```js
function test(a){
  console.log(a)
}
test(10)
```

인자 두개
```js
function sum(a, b){
  return a+b;
}
var c = sum(10,20);
console.log(c);
```

입력과 출력 둘 다 없는 경우
```js
function sum(){
  console.log("sum 함수를 호출했나요");
}
var c = sum();
```

#### 3052
##### 나머지
set는 중복 삭제해서 중복되지 않은 자료형으로 만들어준다~~
```python
arr = []
for i in range(10):
    n = int(input())
    arr.append(n%42)
# print(arr)
arr = set(arr)
print(len(arr))
```
